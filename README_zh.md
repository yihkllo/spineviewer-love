# SpineLove

**SpineLove** 是一款基于Spine 动画查看与导出工具。

支持 Spine 运行时版本：**2.1 · 3.4 · 3.5 · 3.6 · 3.7 · 3.8 · 4.0 · 4.1 · 4.2**

---

## 界面结构

### Select Folder（选择文件夹）
打开文件夹选择器，所选文件夹内找到的所有 Spine 骨骼文件将列在按钮下方。点击列表中的文件即可加载，悬停时显示其父文件夹名称。

---
界面分为**左面板**（文件与播放控制）、**右面板**（尺寸、插槽、队列与导出）以及中间的**画布**预览区。

---

## 左面板

### File（文件）
打开文件选择器，加载 Spine 骨骼文件（`.skel` / `.json` + atlas 图集）。

### Setting（设置）
打开设置对话框，包含三个子页面：

| 子页面 | 说明 |
|---|---|
| **Language（语言）** | 切换界面语言，从内置语言文件中选择（简体中文、繁体中文、日语、韩语、英语）。 |
| **Theme（主题）** | 调整界面外观。包含色相、饱和度、亮度滑条，暗色模式开关，字体大小调整，以及自定义标题栏背景图。 |
| **Render BG Color（渲染背景色）** | 通过颜色选择器设置画布背景颜色，默认为黑色。 |

### Background（背景）
打开文件选择器，加载一张背景图显示在画布动画后面。

### Audio（音频）
打开文件选择器，加载一个或多个音频文件用于同步播放。

加载音频后，出现以下控件：

| 控件 | 说明 |
|---|---|
| **序号 / 总数** | 显示当前音频文件的索引和总数，例如 `1 / 3`。 |
| **文件名** | 显示当前音频文件的名称。 |
| **Play / Stop** | 播放或停止当前音频文件。 |
| **Loop（循环）** | 切换当前音频的循环播放状态，激活时高亮显示。 |
| **Vol 音量滑条** | 调节播放音量，范围 0.0 到 1.0。 |
| **Auto（自动）** | 切换动画时自动播放对应音频，激活时高亮显示。 |
| **`<`** | 切换到已加载列表中的上一个音频文件。 |
| **`>`** | 切换到已加载列表中的下一个音频文件。 |

#### Alpha premultiplied（预乘 Alpha）
切换预乘 Alpha 混合模式的复选框。

#### Offset (0,0) Load（加载时归零偏移）
勾选后，每次加载新文件时将骨骼的绘制偏移量重置为 (0, 0)。

#### Scale（缩放）
滑条范围 10% 至 500%，控制骨骼渲染大小。**Reset** 重置为 100%。

#### Speed（速度）
滑条范围 0× 至 5×，控制动画时间缩放比例。**Reset** 重置为 1.0×。

#### Mix（混合时长）
滑条范围 0 秒至 1 秒，设置动画切换时的默认交叉淡入淡出时长。**Reset** 重置为 0 秒。

---

### Animations（动画列表）
列出已加载骨骼中的所有动画

### Skin / Mix（皮肤 / 混合）
列出已加载骨骼中的所有皮肤。

| 控件 | 说明 |
|---|---|
| **单选模式**（默认） | 点击皮肤即应用，再次点击取消选择（恢复默认皮肤）。 |
| **Mix 复选框** | 启用多皮肤混合模式。勾选任意组合的皮肤，通过 Spine 皮肤混合 API 进行合并显示。 |

---

### `<<`（隐藏面板）
折叠左右两个面板，让画布获得最大空间。将鼠标移到左侧边缘，会出现 **`>>`** 按钮用于恢复面板。

---

### Size/Flip（尺寸/翻转）*可折叠*
显示当前窗口尺寸、骨骼基础尺寸和绘制偏移量。

| 按钮 | 说明 |
|---|---|
| **Mirror（镜像）** | 水平翻转骨骼（切换 X 轴翻转）。 |
| **Rotate（旋转）** | 将骨骼顺时针旋转 90°。 |

---

### Track Mix（轨道混合）*可折叠*
支持在多条 Spine 轨道上同时叠加播放多个动画。

| 控件 | 说明 |
|---|---|
| **动画列表** | 全部动画的多选列表，勾选要添加为轨道的动画。 |
| **Add（添加）** | 将已勾选的动画作为循环轨道添加。 |
| **Clear（清除）** | 移除所有活动轨道并重置列表。 |

---

### Slot（插槽）*可折叠*
管理哪些插槽参与渲染，以及通过鼠标与插槽进行交互。

#### Exclude slot by filter（按过滤词排除插槽）
输入文本过滤词，点击 **Apply（应用）** 将所有名称包含该词的插槽一并排除。

#### Mouse slot hover（鼠标悬停插槽）
| 控件 | 说明 |
|---|---|
| **Enable 复选框** | 启用悬停检测，鼠标移到画布上时高亮显示光标下方的插槽。 |
| **Hovered slot（悬停插槽）** | 显示当前鼠标下方的插槽名称。 |
| **Pinned slot（固定插槽）** | 在悬停状态下点击画布可固定一个插槽，固定后在列表中高亮显示其名称。 |
| **Colour（颜色）** 色块 | 打开颜色选择器，修改高亮轮廓的颜色。 |
| **Slot bounding（插槽边界）** | 固定插槽后，可切换显示其包围盒坐标（X、Y、W、H）。 |

---

### Queue（动画队列）*可折叠*
构建并按顺序播放一组动画。

| 控件 | 说明 |
|---|---|
| **下拉列表** | 从下拉框中选择要加入队列的动画。 |
| **+Add（添加）** | 将选中动画追加到队列末尾。 |
| **Play（播放）** | 从头开始依序播放队列。 |
| **Stop（停止）** | 停止队列播放。 |
| **队列列表** | 显示每条队列动画的序号和时长，当前播放项高亮。 |
| **X**（每行） | 从队列中删除该条目。 |
| **Clear（清除）** | 清空整个队列并停止播放。 |

---

### Export（导出）
切换右侧边缘滑入的**导出浮动面板**。

## 导出面板（右侧浮动）

点击右面板的 **Export** 按钮打开此面板。点击顶部 **`>>`** 关闭。

| 按钮 | 说明 |
|---|---|
| **PNG** | 将当前帧保存为 PNG 图片。 |
| **JPG** | 将当前帧保存为 JPG 图片。 |
| **Alpha ON / OFF** | 切换导出快照是否包含 Alpha 透明通道。 |

### Record（录制）

| 按钮 | 说明 |
|---|---|
| **Export GIF** | 录制当前动画并保存为 GIF 动图。 |
| **Export mp4** | 录制当前动画并保存为 MP4 视频（需要 FFmpeg）。 |
| **Export WebM** | 录制当前动画并保存为 WebM 视频（需要 FFmpeg）。编码进行中时禁用。 |
| **Export PNGs** | *仅限逐动画模式* 将每帧导出为 PNG 序列。 |
| **Export JPGs** | *仅限逐动画模式* 将每帧导出为 JPG 序列。 |

### Export FPS（导出帧率）

| 控件 | 说明 |
|---|---|
| **Image（图像）** | 设置 GIF / PNG / JPG 序列的导出帧率（15–90 fps），支持滚轮调节。 |
| **Video（视频）** | 设置 MP4 / WebM 视频的导出帧率（15–240 fps），支持滚轮调节。 |
| **Free recording ON / OFF（自由录制）** | 在*自由录制*模式（手动开始/停止，任意时长）和*逐动画*模式（自动逐个录制每条动画并分别导出）之间切换。 |

录制或写入帧期间，导出按钮区域会替换为进度条。自由录制模式下显示红色的 **Stop Recording（停止录制）** 按钮。
