# SpineLove

## 功能特性

- 支持 **9 个 Spine 运行时版本**（2.1 至 4.2）的动画预览
- 导出格式涵盖 **PNG / JPG** 单帧、序列帧、**GIF**、**MP4**、**WebM**
- 多皮肤混合、多轨道叠加播放与动画队列管理
- 鼠标悬停插槽检测，支持边界框坐标查看
- 音频同步播放，配合动画一起调试
- 可定制 UI 主题：深色模式、色相/饱和度/亮度调节、字体大小
- 多语言支持：English · 简体中文 · 繁体中文 · 日本語 · 한국어

---

## 界面结构

窗口分为三个区域：

```
┌─────────────┬──────────────────────────┬──────────────┐
│   左面板    │         画布             │   右面板     │
│ （控制区）  │    （动画预览区域）       │  （工具区）  │
└─────────────┴──────────────────────────┴──────────────┘
```

点击 `<<` 可折叠两侧面板，让画布获得最大显示空间。将鼠标移至左侧边缘，点击 `>>` 即可恢复面板。

## 左面板

### File（文件）
打开文件选择器，加载 Spine 骨骼文件（`.skel` / `.json` + atlas 图集）。

### Setting（设置）
打开设置对话框，包含三个子页面：

| 子页面 | 说明 |
|---|---|
| **Language（语言）** | 切换界面语言（简体中文、繁体中文、日语、韩语、英语）。 |
| **Theme（主题）** | 调整界面外观：色相、饱和度、亮度滑条，深色模式开关，字体大小，自定义标题栏背景图。 |
| **Render BG Color（渲染背景色）** | 通过颜色选择器设置画布背景颜色，默认为黑色。 |

### Background（背景）
打开文件选择器，加载一张背景图显示在画布动画后方。按住 `Ctrl` 可控制背景图的位置与缩放。

### Audio（音频）
打开文件选择器，加载一个或多个音频文件用于同步播放。

| 控件 | 说明 |
|---|---|
| **序号 / 总数** | 显示当前音频索引与总数，例如 `1 / 3`。 |
| **文件名** | 显示当前音频文件的名称。 |
| **Play / Stop** | 播放或停止当前音频。 |
| **Loop（循环）** | 切换当前音频的循环播放状态，激活时高亮。 |
| **Vol（音量）** | 调节播放音量，范围 `0.0` 至 `1.0`。 |
| **Auto（自动）** | 切换动画时自动播放对应音频。 |
| **`<` / `>`** | 切换到上一个 / 下一个音频文件。 |

### Alpha premultiplied（预乘 Alpha）
切换骨骼渲染时的预乘 Alpha 混合模式。

### Offset (0,0) Load（加载时归零偏移）
勾选后，每次加载新文件时将骨骼绘制偏移重置为 `(0, 0)`。通常用于含有复杂皮肤的 Spine 文件。

### Scale（缩放）
滑条范围 `10%` 至 `500%`，控制骨骼渲染大小。**Reset** 重置为 100%。

### Speed（速度）
滑条范围 `0×` 至 `5×`，控制动画时间缩放比例。**Reset** 重置为 1.0×。

### Mix（混合时长）
滑条范围 `0 秒` 至 `1 秒`，设置动画切换时的默认交叉淡入淡出时长。**Reset** 重置为 0 秒。

### Animations（动画列表）
列出已加载骨骼中的所有动画，点击即可播放。

### Skin / Skin Mix（皮肤 / 皮肤混合）
列出已加载骨骼中的所有皮肤。

| 模式 | 说明 |
|---|---|
| **单选模式**（默认） | 点击皮肤即应用，再次点击取消（恢复默认皮肤）。 |
| **Mix 模式** | 勾选 Mix 复选框后，可同时选中多个皮肤，通过 Spine 皮肤合并 API 进行混合显示。 |

## 右面板

### Size / Flip（尺寸 / 翻转）
显示当前窗口尺寸、骨骼基础尺寸和绘制偏移量。

| 按钮 | 说明 |
|---|---|
| **Mirror（镜像）** | 水平翻转骨骼（切换 X 轴翻转）。 |
| **Rotate（旋转）** | 将骨骼顺时针旋转 90°。 |

### Track Mix（轨道混合）
在多条 Spine 轨道上同时叠加播放多个动画。

| 控件 | 说明 |
|---|---|
| **动画列表** | 多选列表，勾选要添加为轨道的动画。 |
| **Add（添加）** | 将已勾选的动画作为循环轨道添加。 |
| **Clear（清除）** | 移除所有活动轨道并重置列表。 |

### Slot（插槽）
管理插槽的渲染显示，并通过鼠标与插槽进行交互。

**Exclude slot by filter（按关键字排除插槽）** — 输入文本后点击 **Apply**，将所有名称包含该词的插槽一并排除。

**Mouse slot hover（鼠标悬停检测）** — 开启后可在画布上实时检测插槽：

| 控件 | 说明 |
|---|---|
| **Enable 复选框** | 启用悬停检测，鼠标移到画布上时高亮显示光标下方的插槽。 |
| **Hovered slot（悬停插槽）** | 显示当前鼠标下方的插槽名称。 |
| **Pinned slot（固定插槽）** | 在悬停状态下左键点击画布，可固定一个插槽，并在列表中高亮显示。 |
| **Colour（颜色）色块** | 打开颜色选择器，修改高亮轮廓颜色。 |
| **Slot bounding（插槽边界）** | 固定插槽后，显示其包围盒坐标（X、Y、W、H）。 |

### Queue（动画队列）
构建并按顺序播放一组动画。

| 控件 | 说明 |
|---|---|
| **下拉列表** | 从下拉框中选择要加入队列的动画。 |
| **+Add（添加）** | 将选中动画追加到队列末尾。 |
| **Play / Stop** | 开始或停止队列播放。 |
| **队列列表** | 显示每条动画的序号和时长，当前播放项高亮。 |
| **X**（每行） | 从队列中移除该条目。 |
| **Clear（清除）** | 清空整个队列并停止播放。 |

> **注意：** 导出队列只有在播放状态下才能进行导出操作。

### Export（导出）
打开右侧边缘滑入的**导出浮动面板**。

## 导出面板

点击右面板的 **Export** 按钮打开，点击顶部 **`>>`** 关闭。

所有导出文件均保存在 exe 文件所在的根目录。

### Snapshot（截图）

| 按钮 | 说明 |
|---|---|
| **PNG** | 将当前帧保存为 PNG 图片。 |
| **JPG** | 将当前帧保存为 JPG 图片。 |
| **Alpha ON / OFF** | 切换导出是否保留 Alpha 透明通道。 |

### Record（录制）

| 按钮 | 说明 |
|---|---|
| **Export GIF** | 录制当前动画并导出为 GIF 动图。 |
| **Export MP4** | 录制并导出为 MP4 视频。 |
| **Export WebM** | 录制并导出为 WebM 视频，需将 `ffmpeg.exe` 放置于根目录。编码进行中时禁用。 |
| **Export PNGs** | *仅限逐动画模式* — 将每帧导出为 PNG 序列。 |
| **Export JPGs** | *仅限逐动画模式* — 将每帧导出为 JPG 序列。 |

### FPS（导出帧率）

| 控件 | 范围 | 说明 |
|---|---|---|
| **Image（图像）** | 15–90 fps | GIF / PNG / JPG 序列的导出帧率，支持滚轮调节。 |
| **Video（视频）** | 15–240 fps | MP4 / WebM 视频的导出帧率，支持滚轮调节。 |

### 录制模式

| 模式 | 说明 |
|---|---|
| **Free recording ON（自由录制）** | 手动开始/停止，可录制任意时长。 |
| **Free recording OFF（逐动画）** | 自动逐个录制每条动画并分别导出。 |

录制或写入帧期间，导出按钮区域会替换为进度条。自由录制模式下显示红色 **Stop Recording（停止录制）** 按钮。

## 快捷键

| 快捷键 | 功能 |
|---|---|
| `F11` | 切换全屏模式 |
| `↑` | 选择列表中的上一个 Spine 文件 |
| `↓` | 选择列表中的下一个 Spine 文件 |
| 松开 `↑` / `↓` | 加载当前高亮的 Spine 文件 |
| `←` | 切换到上一个动画 |
| `→` | 切换到下一个动画 |

### 鼠标操作

| 操作 | 功能 |
|---|---|
| 左键单击画布 | 切换到下一个动画 |
| 滚轮 | 缩放当前 Spine 显示大小 |
| `Shift` + 滚轮 | 同时缩放所有可见 Spine 资源 |
| 左键拖拽 | 移动当前 Spine 位置 |
| 左键单击 + 滚轮 | 调整动画播放速度 |
| `Ctrl` + 左键拖拽 | 移动背景图位置 |
| `Ctrl` + 滚轮 | 缩放背景图 |
| 中键单击 | 重置 Spine 缩放并自动适配窗口 |
| 鼠标移入画布 *（悬停模式开启）* | 检测当前光标下方的插槽 |
| 左键单击画布 *（悬停模式开启）* | 固定悬停插槽，并在 Slot 面板中高亮显示 |

## Select Folder（选择文件夹）

打开文件夹选择器，文件夹及子文件夹中的所有 Spine 文件将被自动检索并列在下方。

- **单击**文件即可加载
- **右键**文件可进行操作：添加到收藏夹、打开文件所在目录、多模型加载
- **Add Spine** — 同时加载多个 Spine 资源，可通过右上角可拖动的浮动窗口进行管理

### Favorites（收藏夹）
用于收藏常用文件，以文件路径为索引存储。若文件路径失效，对应收藏条目也会随之失效。
